How It Works

This tool will:

  1. Check if you already have a Stripe Customer ID
  2. Look for active subscriptions associated with your account
  3. If found, it will update your Firebase user record with the Pro status
  4. If no subscription is found but "Force upgrade" is selected, it will upgrade your account anyway

Note: You can find your Firebase User ID in your account settings or by checking the URL when viewing your profile.
